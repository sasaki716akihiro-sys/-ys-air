# 株式会社Y's Air コーポレートサイト

シンプルな静的HTML/CSS/JSサイトです。ビルド不要で、ファイルをそのままレンタルサーバーやNetlify、Vercelなどにアップロードすれば公開できます。

## ファイル構成

```
index.html      … トップページ
business.html   … 事業内容
partners.html   … 協力店募集
company.html    … 会社概要
contact.html    … お問い合わせ
privacy.html    … プライバシーポリシー
css/style.css   … 全ページ共通のデザイン（配色・余白・文字サイズなど）
js/main.js      … メニュー開閉・後述の設定値の反映
js/site-config.js … 後から変更する情報をまとめた設定ファイル
assets/         … ロゴ画像などを置く場所
```

## 後から情報を追加・変更する方法

### 1. ロゴ画像を設定する

`assets/logo.png` にロゴ画像ファイルを置いてください。ファイルを置くだけで、ヘッダーとトップページに自動的に表示されます（ファイルが無い間は会社名のテキストのみ表示されます）。

### 2. 電話番号・営業時間・定休日・古物商許可番号を追加する

`js/site-config.js` を開き、わかっている項目に値を入力してください。

```js
window.SITE_CONFIG = {
  phone: "078-000-0000",                          // 電話番号
  businessHours: "9:00〜18:00",                   // 営業時間
  closedDay: "日曜・祝日",                         // 定休日
  licenseNumber: "兵庫県公安委員会 第000000000号", // 古物商許可番号
  googleFormUrl: "https://forms.gle/xxxxxxxx",     // お問い合わせ用Googleフォーム
};
```

保存すると、トップページ・会社概要ページ・お問い合わせページに自動で反映されます。空欄のままの項目は「準備中」「取得後掲載予定」と表示されます。

### 3. Googleフォームを設定する

`gas/create-form.gs` のスクリプトを使うと、下記の項目を持つフォームと回答用スプレッドシートを自動作成できます。

1. https://script.google.com/ で新しいプロジェクトを作成
2. `gas/create-form.gs` の内容をすべて貼り付ける
3. 関数「createYsAirContactForm」を選択して実行（初回は権限の承認が必要）
4. 実行ログに表示される「フォーム公開用URL」をコピー
5. `js/site-config.js` の `googleFormUrl` に貼り付ける

手動で作成する場合は、以下の項目で「株式会社Y's Air お問い合わせフォーム」を作成し、フォームのURLを `js/site-config.js` の `googleFormUrl` に貼り付けてください。

**必須項目**
- お問い合わせ種別（プルダウン）
  - 空調メンテナンスについて
  - 中古品販売について
  - 中古品買取について
  - 協力店募集について
  - その他
- お名前
- 会社名・屋号
- メールアドレス
- 電話番号
- お問い合わせ内容
- 個人情報の取り扱いへの同意

**協力店募集の方のみ（任意項目）**
- 対応可能エリア
- 対応可能業務
- 法人・個人事業主の別
- 保有資格・許認可
- HP・SNS・会社URL

回答はGoogleスプレッドシートに連携して管理できます（フォームの「回答」タブから設定）。

### 4. その他の修正

- 文言・会社情報は各HTMLファイル内に直接記載されています。変更したい箇所をテキストとして検索して編集してください。
- サイト公開後は、各HTMLファイル内の `og:url`（例: `https://example.com/`）を実際のドメインに変更してください。

## デザインを調整したい場合

配色・余白・文字サイズ・カードのデザインなどは `css/style.css` の冒頭にある `:root` の変数（`--color-navy` など）や各セクションのスタイルを編集してください。
